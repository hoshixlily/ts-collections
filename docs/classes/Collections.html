  • binarySearch<TElement>(sequence: IList<TElement> | TElement[], element: TElement, comparator?: OrderComparator<TElement, TElement>): number

  • Performs a binary search on the given sequence and returns the index of the element. The sequence must be sorted prior to calling this method. If the searched element exists multiple times in the sequence, the returned index is arbitrary.

    Returns

    The index of the element that is equal to the searched element. If the element is not found, returns -1.

    Type Parameters

    Type Parameters

    Parameters

    • element: TElement

      The element that will be searched for.

    • Optional comparator: OrderComparator<TElement, TElement>

      The comparator method that will be used to compare the elements. It should always be provided if the sequence is of a complex type.

Static rotate

  • rotate<TElement>(sequence: IList<TElement> | TElement[], distance: number): void

  • Rotation of the elements in a given sequence.

    Example:

    • If the sequence is [1, 2, 3, 4, 5] and the distance is 2, the result is [4, 5, 1, 2, 3].
    • If the sequence is [1, 2, 3, 4, 5] and the distance is -2, the result is [3, 4, 5, 1, 2].

    Type Parameters
